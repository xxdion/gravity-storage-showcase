# Mac 一键运行版说明

## 使用方法

1. 解压压缩包。
2. 打开 `gravity-storage-showcase` 文件夹。
3. 双击 `start-mac.command`。
4. Safari 会自动打开：

```text
http://127.0.0.1:5174/
```

登录账号：

```text
账号：siat-admin
密码：demo123456
```

## 注意事项

- 这个一键运行版主要适用于 macOS。
- 第一次打开时，如果系统提示“无法打开未识别开发者文件”，可以右键点击 `start-mac.command`，选择“打开”。
- 如果老师电脑没有安装 Node.js 或 Python，建议使用源码版按 `README-运行说明.md` 安装环境。
- 关闭启动窗口会同时停止前端和后端服务。
